Heuristique : extraction par logique

Heuristique Description -> lignes du tableau
Domiciliation -> IBAN
bloc avant Facture -> vendeur
bloc après vendeur -> client

Fonctionnement

  • une heuristique utilise le contexte, l'ordre et la position
  • elle reconstruit la donnée même si le texte varie
  • elle sert quand une regex seule serait trop fragile

Utilisée surtout pour

  • RIB
  • Facture
  • Devis
+ utile quand la mise en page n'est pas toujours la même
+ plus souple face aux erreurs OCR et aux documents semi-structurés
+ complète les regex sur les zones variables
